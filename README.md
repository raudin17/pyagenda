# Py Agenda


Py Agenda es un **Administrador de contactos🌟** hecho en python **sin dependencias👏** y es compatible con **Python 3🍻**.

## Instalacion

```bash
$ sudo snap install --channel=edge pyagenda --devmode
```

## Uso

Solo ejecuta lo siguiente en la terminal:

```bash
$ pyagenda
```
